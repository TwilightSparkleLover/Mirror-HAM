void* ham_Malloc (  u32    num_bytes ) 
 

A more verbose malloc extension

Parameters:
num_bytes  Number of bytes you want to allocate in the GBA's EXWRAM heap
This is just the malloc() function, but it will give you an error screen when an allocation fails. Nothing more, just for more clarity while debugging.

See also: ham_GetAvailableHeap()
