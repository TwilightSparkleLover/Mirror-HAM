Detailed Description

This set of functions allows to easily poll the Control Pad's different buttons. It relies on a function to be regularly called, which updates the status of the various buttons. The System allows for checking both if a button was recently pressed, or if it is "Held" (continously pressed).

Here is a little example program: 

void myFunction()
{
    // Initialize 
    ham_InitPad();

    // Infinite loop
    while(TRUE)
    {
        // Make sure its a new frame
        if(IsNewFrame)
        {
            // Read pad states
            ham_UpdatePad();

            // Access the Pad variable
            // Check if button down has been pressed
            if(Pad.Pressed.Down)
            {
                Go_One_Menu_Item_Down();
            }
            else
            if(Pad.Pressed.Up)
            {
                Go_One_Menu_Item_Up();
            }

            IsNewFrame = FALSE;
        }
    }

    // reset pad system
    ham_ResetPad();
}

Update pad states.

This should be called once per frame.
